# Workshop iOS - WebTech PUC MG

### Descrição
O projeto consiste em um app iOS nativo para ser apresentado no Workshop de iOS do WebTech PUC MG - Mobile Days.

### Tecnologias Utilizadas
- **SwiftUI**: Framework para desenvolver a interface de usuário.
- **Combine**: Utilizado para gerenciamento de dados reativos e comunicação entre UI e modelo de dados.
- **MapKit**: Utilizado para desenvolver mapas.
- **UserDefaults**: Utilizado para persistência de dados localmente.

### Configuração do Projeto
Para configurar o projeto em sua máquina local, siga os passos abaixo:

1. Clone o repositório:
   ```git clone https://github.com/seu-usuario/CheckpointView.git```
2. Abra o projeto:
   ```open CheckpointView.xcodeproj```
